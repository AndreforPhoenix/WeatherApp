About Project

I began developing this data tool as a final project for a software development course I took through Course Careers during the second half of 2024. This application consists of a frontend (HTML & JS), a backend (Java (REST API)), and a relational database (MySQL). The application was deployed using Azure app service with a custom domain.

About Dataset

The source of the data that the application uses to generate graphical infomation and statistics is produced and managed by the The Arizona Meteorological Network.
